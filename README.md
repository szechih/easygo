## ⚡ 简介

Easy Go， 基于 `React18`,`TypeScript`,`webpack5`,`antd-mobile`等相关主流技术开发，一个免费开源的移动端网页购物平台，也可适用于学习`React`时进行参考或练手的项目。

## 🔨 安装

-   克隆项目

```bash
git clone https://github.com/szechih/easygo.git
```

-   安装依赖

```bash
cd easygo

cnpm install
yarn install
//安装失败，请升级 nodejs 到 16 以上
```

-   运行项目

```bash
npm run dev
yarn dev
```

-   打包项目

```bash
npm run build
yarn build
```
